# E-Learning Website using HTML5, CSS3, Bootstrap5 and JavaScript 

![E-Course Website](preview.jpg)

The project involves the creation of an e-learning website using HTML, CSS, Bootstrap, and JavaScript. The website aims to provide a user-friendly and responsive platform for learners to access educational content. 


## Introduction

Secret Coder is an e-learning website designed to provide a user-friendly and responsive platform for learners to access educational content. The project incorporates HTML, CSS, Bootstrap, and JavaScript to create an interactive and visually appealing learning experience.

## Features

- Responsive design using Bootstrap.
- Structured course catalog with detailed descriptions.
- Interactive lessons.
- User can register and login.
- User authentication for personalized learning experiences.
- Instructor application form.
- User can also contact us.
